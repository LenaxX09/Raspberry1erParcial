# P10 - OLED con Sensor DHT11

## Descripción
Esta práctica combina el sensor DHT11 con una pantalla OLED para crear un monitor ambiental mejorado con mejor resolución y visualización.

## Objetivos
- Integrar sensor DHT11 con OLED
- Crear interfaces gráficas de datos
- Mostrar información de forma atractiva
- Implementar actualización periódica eficiente

## Requisitos
- Raspberry Pi
- Pantalla OLED (128x64)
- Sensor DHT11
- Librerías Adafruit para OLED y DHT

## Contenido
- **P10-OLED-DHT11.odt**: Documentación completa
- **P10_dht11_oled/**: Código del proyecto

## Características
- Visualización en alta resolución
- Gráficos y símbolos personalizados
- Histórico de temperatura
- Indicadores visuales de condiciones ambientales

## Diseño de Interfaz
```
┌──────────────────┐
│ Temperatura: 25°C│
│ Humedad: 60%     │
│ Estado: Normal   │
└──────────────────┘
```

## Mejoras Respecto a LCD
- Mayor espacio para información
- Posibilidad de gráficos más detallados
- Mejor legibilidad visual
